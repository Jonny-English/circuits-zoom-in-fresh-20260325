# Pre-P4 基础包

这组基础 lab 是给还没稳定进入文章主线的读者准备的。目标不是“再看几篇论文”，而是先补齐四个会真正卡住小白的缺口：

- 环境和 baseline 纪律
- attention 形状感
- 向量几何与 feature 语言
- sweep、ablation 和 failure analysis

## 建议顺序

1. [F00 环境、图表与基线纪律](f00-environment-plots-baselines.md)
2. [F01 Transformer 形状与注意力读图](f01-transformer-shapes-attention.md)
3. [F02 向量几何、特征与探针](f02-vector-geometry-features-probes.md)
4. [F03 Sweep、Ablation 与 Failure Analysis](f03-sweeps-ablations-failure-analysis.md)

## 什么时候从这里毕业

- 你能稳定跑 notebook，而不是靠碰运气刷新环境。
- 你能分清 baseline、variant、sweep、ablation 分别在干什么。
- 你读图时能说出 shape、direction 和 metric，而不是只会说“这里好像有点变化”。
- 你已经愿意把实验写成 log，而不是只在脑子里记一个印象。
