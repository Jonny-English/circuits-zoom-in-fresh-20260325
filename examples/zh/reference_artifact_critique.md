# 参考 Artifact Critique

## Artifact

`M09` 的 introspection mismatch 图。

## 这个 artifact 支持什么

- 它支持“self-report 与行为信号并不总是一致”。
- 它支持“最值得继续研究的，往往是 gap 最大的案例”。
- 它支持“只看模型怎么描述自己，不足以推出内部状态结论”。

## 这个 artifact 不支持什么

- 它不支持“模型已经拥有稳定内省能力”。
- 它不支持“gap 大就一定说明模型在隐藏真实状态”。
- 它不支持“当前教学数据里的 mismatch 会在真实开放任务里同样出现”。

## 还缺哪条证据链

缺的是跨任务、跨 prompt 分布、跨时间的一致性证据。如果 mismatch 只在单一任务里出现，那更像局部行为现象，而不是稳定内部机制。

## 下一步实验

把当前 mismatch 最大的两个案例放到三种新分布里重测：更长上下文、更高压约束、以及需要显式规划的任务。核心问题是：gap 是否稳定，还是会随任务形状快速塌掉？
