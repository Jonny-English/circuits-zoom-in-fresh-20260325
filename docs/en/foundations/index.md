# Pre-P4 Foundation Pack

These labs are for readers who are not yet stable enough to enter the article-first path. The goal is not "read more papers." The goal is to repair four gaps that actually block beginners:

- environment setup and baseline discipline
- attention-shape intuition
- vector geometry and the feature language
- sweeps, ablations, and failure analysis

## Recommended order

1. [F00 Environment, Plots, and Baseline Discipline](f00-environment-plots-baselines.md)
2. [F01 Transformer Shapes and Attention Reading](f01-transformer-shapes-attention.md)
3. [F02 Vector Geometry, Features, and Probes](f02-vector-geometry-features-probes.md)
4. [F03 Sweeps, Ablations, and Failure Analysis](f03-sweeps-ablations-failure-analysis.md)

## When to graduate from this pack

- You can run notebooks reliably instead of succeeding by luck.
- You can explain what a baseline, variant, sweep, and ablation each do.
- When you read a plot, you can name the shape, direction, and metric instead of saying only "something changed here."
- You are willing to write experiments as logs instead of keeping only an impression.
