## 使用规则

- 只有运行时实时产出结果的 notebook 才算 runnable。
- `reading-only` 的意思是保留讲义，但 notebook 故意不生成。
- `X01-X07` 是可运行的方法练习，不冒充完整论文级复现。
- 如果这个仓库帮你找到了更清晰的路径，就给它一个 star，让更多小白找到它。
