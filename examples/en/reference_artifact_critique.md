# Reference Artifact Critique

## Artifact

The introspection-mismatch plot in `M09`.

## What this artifact supports

- It supports the claim that self-report and behavioral signals do not always align.
- It supports the idea that the most informative cases are often the largest gaps.
- It supports the warning that self-description alone is insufficient for stronger claims about internal state.

## What this artifact does not support

- It does not support the claim that the model already has stable introspective ability.
- It does not support the claim that a large gap necessarily means the model is hiding a true internal state.
- It does not support the claim that mismatches in this teaching data will appear the same way in open real-world tasks.

## Which evidence chain is still missing

The missing chain is consistency across tasks, prompt distributions, and time. If the mismatch appears only in one narrow task frame, it looks more like a local behavior than a stable internal mechanism.

## Next experiment

Take the two largest-gap cases and re-test them in three new distributions: longer contexts, higher-pressure constraints, and tasks that require explicit planning. The core question is whether the gap stays stable or collapses quickly as the task shape changes.
