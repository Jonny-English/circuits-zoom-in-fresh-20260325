# M08 Persona Vectors

## Goal

Connect feature-style thinking to character traits by treating persona as a direction in representation space.

## Core reference

- [Persona Vectors](https://www.anthropic.com/research/persona-vectors)

## What to look for

- Character can be monitored as movement along a direction.
- Trait shifts can be compared before and after intervention.
- A persona vector is a useful handle, not a complete theory of character.

## Notebook and artifacts

- Notebook: `notebooks/en/m08_persona_vectors.ipynb`
- Shared artifact: `artifacts/m08_persona_vectors.json`

## Self-check questions

- Which character properties become newly measurable or comparable once you have persona vectors?
- Even if a vector direction works in the current artifact, where could persona control still drift or destabilize?
- If you had to test the stability of persona control, how would you evaluate it across prompts, tasks, or time?

## Takeaway

Interpretability starts touching model character once representation directions become behaviorally meaningful.
